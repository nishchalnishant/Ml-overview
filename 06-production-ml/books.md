# Books

Recommended reading for production ML and MLOps. Ordered by how foundational they are.

---

## Designing Machine Learning Systems

**Author:** Chip Huyen
**Best for:** End-to-end mental model of production ML — data, features, training, serving, monitoring, and drift — written for practitioners who ship.
**Book notes:** [../10-references/book-notes/mlops/designing-machine-learning-systems.md](../10-references/book-notes/mlops/designing-machine-learning-systems.md)

---

## Machine Learning Engineering

**Author:** Andriy Burkov
**Best for:** Practical, dense engineering patterns for building and deploying real ML systems — less theory, more "here is what breaks and how to fix it."
**Book notes:** [../10-references/book-notes/mlops/machine-learning-engineering.md](../10-references/book-notes/mlops/machine-learning-engineering.md)

---

## Building Machine Learning Powered Applications

**Author:** Emmanuel Ameisen
**Best for:** Product-minded ML — how to go from idea to deployed application, with strong coverage of the iteration loop between data, model, and user feedback.
**Book notes:** Not yet available

---

## Reliable Machine Learning

**Author:** Cathy Chen, Niall Richard Murphy, Kranti Parisa, D. Sculley, Todd Underwood et al.
**Best for:** Operating ML systems in production with SRE discipline — reliability, monitoring, incident response, and organizational patterns.
**Book notes:** Not yet available

---

## Machine Learning Design Patterns

**Author:** Valliappa Lakshmanan, Sara Robinson, Michael Munn
**Best for:** Catalog of reusable design patterns for data representation, problem framing, model training, and serving — reference-style, dip in when needed.
**Book notes:** [../10-references/book-notes/mlops/machine-learning-design-patterns.md](../10-references/book-notes/mlops/machine-learning-design-patterns.md)
