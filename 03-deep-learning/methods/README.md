# Deep Learning Methods

This folder is about where deep learning shows up in the real world.

Not just:

- "what is a Transformer?"

But:

- why it wins in NLP
- when CNNs still make more sense
- why generative models feel magical and difficult at the same time
- how time-series modeling changes the rules

Think of this as the **use-case layer** on top of the core deep-learning parts.

## How To Read It

A strong deep-learning answer should usually do three things:

1. explain the model idea
2. explain why it fits the data type
3. explain the production tradeoff

That last one matters.

Because in the real world, accuracy is only part of the story.
