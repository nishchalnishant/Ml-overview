# PyTorch — Content Index

This folder has two PyTorch files. They cover different ground and are meant to be read in order.

---

## Files

### [pytorch-foundations.md](pytorch-foundations.md)

Conceptual introduction that bridges DevOps/systems thinking to PyTorch's mental model. Covers tensors, autograd, and the training loop framed as "tensors are the payload, autograd is the dependency tracker, the training loop is your build pipeline." Start here if PyTorch is new to you or if you want the why before the how.

### [pytorch-fundamentals.md](pytorch-fundamentals.md)

Dense practical reference organized the way interviews actually probe the topic: tensors, autograd, model definition, the training loop, architecture patterns, and production/debugging details most people skip. Includes code for everything. Read this when you need to write PyTorch from scratch or debug a training issue.
