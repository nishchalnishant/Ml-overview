# Emerging Topics

Frontier and cross-cutting topics that don't belong to a single classical track: 2023–2025 model trends, the responsible-AI cluster (fairness, privacy, interpretability, robustness), causal inference, and continual learning. These come up in senior and research-leaning interviews and move fast — treat them as a current snapshot rather than settled canon.
