## Deep Learning

Notes in [deep-learning/](deep-learning/)

| Book | Read if you want... |
|------|---------------------|
| [Alice in Differentiable Wonderland](deep-learning/01-alice-in-differentiable-wonderland.md) | An intuitive, visual introduction to deep learning fundamentals without heavy math |
| [Build a Large Language Model from Scratch](deep-learning/02-build-a-large-language-model-from-scratch.md) | To implement a GPT-style LLM step by step and understand every component from the ground up |
| [Deep Learning: A Practitioner's Approach](deep-learning/03-deep-learning-practitioner-approach.md) | A hands-on reference covering neural network patterns and DL4J-style applied deep learning |
| [Deep Learning with PyTorch](deep-learning/04-deep-learning-with-pytorch.md) | A solid ground-up introduction to deep learning using PyTorch with real worked examples |
| [Dive into Deep Learning](deep-learning/05-dive-into-deep-learning.md) | Rigorous mathematical treatment alongside runnable code across CNNs, RNNs, and Transformers |
| [Grokking Deep Learning](deep-learning/06-grokking-deep-learning.md) | To understand how neural networks actually work by building everything from scratch with numpy |

**Other DL books worth knowing (not yet summarized):**

| Book | Why it matters |
|------|----------------|
| Deep Learning (Goodfellow, Bengio, Courville) | The canonical DL textbook — theory-heavy, free online |
| Understanding Deep Learning (Simon J.D. Prince) | Modern DL textbook with recent content (diffusion, LLMs, flow models) — free PDF |
| The Little Book of Deep Learning (François Fleuret) | Concise but rigorous; fits in a pocket; great for review |

## Machine Learning

Notes in [machine-learning/](machine-learning/)

| Book | Read if you want... |
|------|---------------------|
| [Machine Learning Pocket Reference](machine-learning/machine-learning-pocket-reference.md) | A fast, dense reference for classical ML workflows, preprocessing, and model selection |

**Other ML books worth knowing (not yet summarized):**

| Book | Why it matters |
|------|----------------|
| The Elements of Statistical Learning (Hastie, Tibshirani, Friedman) | The rigorous theoretical foundation for classical ML — free PDF at ESL website |
| An Introduction to Statistical Learning (James et al.) | Accessible version of ESL with R/Python labs — ISLP edition has Python code |
| Hands-On Machine Learning (Aurélien Géron) | Best practical scikit-learn + Keras + TF reference for working ML engineers |
| Pattern Recognition and Machine Learning (Bishop) | Bayesian perspective on ML; Gaussian processes, graphical models, EM algorithm |
