## Which to read first

If you are new to deep learning: start with **Grokking Deep Learning**, then **Deep Learning with PyTorch**.

If you already understand the basics and want LLM internals: go straight to **Build a Large Language Model from Scratch**.

If you want rigorous treatment with working code: **Dive into Deep Learning** is the most comprehensive single resource here.
