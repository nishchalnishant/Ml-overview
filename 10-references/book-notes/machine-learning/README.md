# Machine Learning Book Notes

## Machine Learning Pocket Reference

**Notes:** [machine-learning-pocket-reference.md](machine-learning-pocket-reference.md)

A fast, dense reference covering classical ML workflows end-to-end: data cleaning, preprocessing, feature engineering, model selection, evaluation, and interpretation.

**Who should read it:** Engineers who already understand ML concepts but want a reliable quick-reference during active project work — the kind of book you keep open in a tab rather than read cover-to-cover. Especially useful if you work across many different problem types and need to quickly recall the right tool or technique.

Not the right book if you want deep theory or coverage of deep learning. For that, see [../deep-learning/](../deep-learning/).
