# MLOps Book Notes

Six books, different angles on production ML. Pick based on your current pain point.

---

| Book | Best for | Notes |
|------|----------|-------|
| [Designing Machine Learning Systems](designing-machine-learning-systems.md) | Full mental model for production ML — data, features, training, serving, monitoring | [notes](designing-machine-learning-systems.md) |
| [Machine Learning Design Patterns](machine-learning-design-patterns.md) | Reusable patterns for data representation, training, and serving decisions | [notes](machine-learning-design-patterns.md) |
| [Machine Learning Engineering](machine-learning-engineering.md) | Dense practical patterns for building and operating real ML systems | [notes](machine-learning-engineering.md) |
| [Keras to Kubernetes](keras-to-kubernetes.md) | End-to-end deployment: from training to container orchestration | [notes](keras-to-kubernetes.md) |

---

## Which to read first

If you need a full production ML mental model: start with **Designing Machine Learning Systems** (Huyen). It is the most comprehensive single resource here.

If you want reusable design patterns for specific decisions: **Machine Learning Design Patterns** (Lakshmanan et al.).

If you are deploying a specific model to infrastructure: **Keras to Kubernetes**.

## Additional books worth knowing (not yet summarized)

| Book | Why it matters |
|------|----------------|
| Introducing MLOps (Treveil et al.) | Fundamentals of MLOps lifecycle — bias, fairness, explainability |
| Machine Learning Engineering with Python (Fareri) | Python-first MLOps with coverage of Generative AI and LangChain |
| Reliable Machine Learning (Murphy et al.) | SRE mindset applied to ML — reliability, accountability, monitoring |
| Building Machine Learning Pipelines (Hapke & Nelson) | TensorFlow-based pipeline automation with Airflow and Kubeflow |
| Practical MLOps (Gift & Deza) | AWS, Azure, GCP deployment and tool selection guide |
| Machine Learning Engineering in Action (Bärtsch) | Design patterns for secure, scalable ML — logging and troubleshooting |
| Implementing MLOps in the Enterprise (Raj et al.) | Production-first: continuous operational pipelines for organizations |
| Engineering MLOps (Morissette) | Azure-focused; MLOps techniques for scalable ML lifecycles |
